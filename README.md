# betterprompt
Better Prompt helps creators, students, and developers craft better AI instructions.
It automatically organizes your input into role, context, constraints, and tone — producing concise prompts that yield better model responses.
✨ Features
AI-powered prompt refinement (via OpenAI API)
Structured input fields and templates
Copy, download, or customize outputs instantly
Runs directly in your browser — no build step
